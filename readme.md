# School v0.1

## Start guide
⚠ *Для работы с проектом на компьютере требуется иметь 
установленный Docker и включенную виртуализацию в BIOS.*

> Start docker server on localhost:8080  
> $ docker compose up

> Stop docker server  
> $ docker compose down
